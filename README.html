[Worklets]

Category(ies): Evolution, Adaptors/Wrappers, Configuration Management, Dynamic Analysis & Tuning, Programming Environments, Information Management, Network Management
Institution/Company: Columbia University Programming Systems Lab
Description: The Worklets system is designed a mobile code infrastructure used for dynamic re-configuration of third-party systems. Instead of bringing down entire components of a distributed system at runtime for recompilation, worklets [intelligent code agents] can be injected transparently into the system for incremental and [persistent] re-structuring.
For more information: worklets@cs.columbia.edu, http://www.psl.cs.columbia.edu/
Assumptions: System Requirement: Java 1.3
Status: Research Prototype
Availability: Get library and documentation from http://www.psl.cs.columbia.edu/~gskc/psl/
